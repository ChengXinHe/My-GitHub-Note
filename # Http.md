* **Form**
  ```
  <form action="" method="">
  <input type="" name="" value="">
  <label for="">
  <button>
  </form>
  ```
  >for和id对应，可以按下label全选input
  >input必须拥有name属性，这样才可以成功提交
  >post方法和get方法：post后url不会显示提交的内容
  >button必须在form内才有效
  >input类型:checkbox, radio, range, text, number, email, file, password
  ><option> 可以增加一个空白选项以达到初始状态为空的效果
